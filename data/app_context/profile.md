# Profile

## Accessing Your Profile
- All users can access their own profile from the main navigation menu.
- The profile page shows: first name, last name, profile picture, email, role, and organization.

## Editing Your Profile
- Users can edit their own first name, last name, and profile picture.
- Email is read-only and cannot be changed.
- Admins can also edit other users' profiles.

## Changing Your Password
- Users can change their password from their profile page.
- They must provide their current password and the new password.

## Profile Stats
- Students see mini stats: courses completed, tests taken.
- Teachers see mini stats: courses created, classes in scope.
- Admins see basic profile info without detailed stats.

## Common Issues
- Can't change email: email is read-only on the platform.
- Forgot current password: use the forgot password flow from the login page.
- Profile picture not updating: try refreshing the page after saving.
- Want to change your name: go to your profile from the navigation menu and edit it there.
