# Tests & Attempts

## Test Types
- FIXED (stable): predefined tests created by the teacher and added to the course content tree.
- GENERATED: AI-generated tests created on demand based on selected course content.

## Taking a Stable Test
- Students open a stable test from the course content tree on the Course Study Page.
- They are taken to the Test Runner Page to answer questions.

## Generating a Personalized Test
- Students select one or more content nodes from a course on the Course Study Page.
- They go to the Test Generation Page where they confirm the selected topics.
- They choose the number of questions and optionally set a timer.
- The AI generates the test and the student starts the Test Runner.

## Test Runner
- Questions are shown one at a time.
- Students select an answer for each question and can navigate between questions.
- A timer is shown if the test has one.
- Students submit the test at the end to get their score.
- Answers are saved locally during the session and sent to the backend only on final submit.

## Test Results
- After submitting, students see their score and a review of each question.
- The review shows the student's answer, the correct answer, and whether it was correct.
- For AI-generated tests, students can report a question they believe is incorrect.

## AI Test Generation (Teachers)
- Teachers can generate tests from course content using AI from the Test Editor / AI Generate Test page.
- They select which content nodes to base the test on and how many questions to generate.
- The AI produces a draft which the teacher can review, edit, and save as a stable test in the course.
- Teachers can regenerate individual questions or the entire test.

## Attempts
- Each submitted test creates an attempt.
- Students can view their attempt history per course on the Course Progress Page.
- Each attempt shows: score, date, test type.
- Students can open a past attempt to review their answers.

## Common Issues
- Test not loading: refresh the page or check your internet connection.
- Score not showing: make sure the test was submitted, not just saved.
- Can't start a test: the course may not be assigned to your class or the test may not be published.
- Timer ran out: the test is automatically submitted when the timer expires.
- Want to report a wrong question: use the report button on the Test Results page (only for AI-generated tests).
