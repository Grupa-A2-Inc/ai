# Classes & Organizations

## Classes Overview
- Classes (class groups) belong to an organization.
- Each class has a name, optional description, and academic year.
- Students are assigned to one class.
- Teachers are associated with classes through their assignment scope.

## For Admins — Classes Page
- Admins see all classes in their organization.
- Admins can create new classes (name, academic year, description).
- When creating a class, teachers can optionally be assigned right away.
- From the Classes Page, admins navigate to the Class Management Page for each class.

## Class Management Page
- Admins can edit class details (name, academic year, description).
- Admins can view the list of students in the class.
- Admins can add available students to the class.
- Admins can remove students from the class.
- A student belongs to only one class.

## Teacher Assignment Scope
- Each teacher has an assignment scope set by the admin:
  - ALL_CLASSES: the teacher can assign their courses to any class in the organization.
  - SELECTED_CLASSES: the teacher can only assign their courses to specific classes chosen by the admin.
- This scope is configured in the User Management page, not in the class management page.

## For Students
- Students see the classes they are enrolled in.
- Course access depends on class assignments — a course assigned to your class will appear in your course list.

## Common Issues
- Course not visible: check if the course is assigned to your class.
- Can't enroll in a class: contact your organization admin.
- Wrong class assignment: contact your organization admin.
- Teacher can't assign course to a class: the teacher may have SELECTED_CLASSES scope and that class is not in their allowed list — contact the admin.
