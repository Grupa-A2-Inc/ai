# Platform Overview

## What is this platform?
An AI-powered e-learning platform called TestifyAI for students, teachers, and organizations.

## User Roles
- STUDENT: accesses assigned and public courses, takes tests, tracks own progress.
- TEACHER: creates and manages courses, assigns courses to classes, views student progress.
- ORG_ADMIN: manages users (students and teachers) and classes within their organization.
- PLATFORM_ADMIN: full platform access across all organizations.

## Main Features
- Authentication: login, register organization, forgot/reset password, profile management.
- Courses: browse, create, publish, archive. Structured as trees of sections and resources (PDF, video, text, links). Teachers assign courses to classes.
- Tests: stable tests (created by teachers) and AI-generated tests (on demand from course content). Students take tests, submit answers, get scores and reviews.
- Progress: track course completion percentage and topic mastery scores per student.
- Classes: group students. Courses are assigned to classes. Teachers have an assignment scope (all classes or selected classes).
- Admin panel: manage users, roles, classes, and organization info.
- Profile: view and edit personal info, change password.

## Navigation by Role
- Student main page: Courses Page (assigned, public, archived tabs).
- Teacher main page: My Courses Page.
- Admin main page: Admin Dashboard.

## Common Issues
- Can't log in: check email/password, account must be ACTIVE.
- Course not visible: must be assigned to your class or set to PUBLIC and PUBLISHED.
- Progress not updating: complete resources and submit tests (not just save).
- Teachers/students can't self-register: accounts are created by the organization admin.
- Contact support for: password reset, account activation, role changes, billing.
